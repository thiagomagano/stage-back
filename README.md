## Description

Este é o repositório backend do Case Técnico da Stage.

## Instalação

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Rodar docker com o banco

Certifique de que tem o docker instalado na máquina:

$ docker compose up
